# Claude Docker Dev Environment
Creates a container to run yoru claude code on.
Automatically clones your repository.

## Warning 
Be aware that this container mounts your .ssh, meaning the container can use your SSH identity.

## Installed dependencies
- Dotnet
- Node JS
- Git

## Usage
# Environment Variables
Copy .env.example files and rename to .env
Edit the file with your configs. 

# Start Container
docker compose up

# Enter container
docker compose exec claude bash
